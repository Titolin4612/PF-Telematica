✨ Bienvenido al Servicio Telemático

Este servicio web ligero, escalable y profesional utiliza Python + Flask para el backend, HTML + CSS para una interfaz moderna, y Docker para un despliegue eficiente en entornos locales o en la nube. Descubre sus características y lánzalo en minutos.

🌐 Cómo Desplegar el Servicio

Sigue estos pasos para ejecutar esta aplicación en tu máquina local o servidor en la nube:

  1. Clona el repositorio oficial del proyecto: git clone https://github.com/Titolin4612/PF-Telematica.git
  2. Entra al directorio del proyecto y Construye la imagen Docker desde el Dockerfile: docker build -t servicioweb:v1 .
  3. Ejecuta el contenedor en segundo plano: docker run -d -p 80:80 servicioweb:v1
  4. Abre tu navegador y visita: http://localhost o http://ip publica

¿Quieres desplegarlo en la nube? Sigue la guía detallada en el README del proyecto para desplegar en AWS EC2 u otras plataformas cloud.
